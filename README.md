# NPOC Ciência (Frontend)

Projeto **apenas frontend** em Next.js (App Router) + Tailwind.

## Requisitos
- Node.js v24+ (testado com v24.12.0)
- npm

## Rodar
```powershell
npm install
npm run dev
```

Se o Turbopack der qualquer dor de cabeça, rode com Webpack:
```powershell
npm run dev:webpack
```

## Login (mock)

## Observação (Windows)
Se você baixou um `.zip` e o comando `unzip` não existe:
```powershell
Expand-Archive -Path .\npoc-ciencia-frontend.zip -DestinationPath .\npoc-ciencia-frontend
```
